# 환경변수

> **누가 봄**: DevOps, BE
> **언제 봄**: 첫 배포, 새 키 추가, 키 회전
> **단일 진실의 원천**: Railway 대시보드 (실제 값), 이 문서 (의도/획득 방법)

## 데이터베이스 (Railway 자동 주입)

| 키 | 자동 주입 | 설명 |
|---|---------|------|
| `PGHOST` | ✅ | PostgreSQL host (보통 .railway.internal) |
| `PGPORT` | ✅ | 보통 5432 |
| `PGDATABASE` | ✅ | DB 이름 |
| `PGUSER` | ✅ | 사용자명 |
| `PGPASSWORD` | ✅ | 비밀번호 |

`application.yml`에서 `${PGHOST:localhost}` 형식으로 폴백 (로컬 개발).

PG add-on 추가하면 main 서비스에 자동으로 위 변수가 노출됨. 별도 설정 X.

## 애플리케이션 (수동)

| 키 | 필수 | 기본값 | 설명 |
|---|------|------|------|
| `PORT` | Railway 자동 | 8092 | 리스닝 포트 |
| `JWT_SECRET` | ⚠️ 필수 (운영) | dev-secret | JWT 서명 키. **32+ chars 랜덤** |
| `OTAKU_CORS_ALLOWED_ORIGINS` | 선택 | `*` | CORS allow origin |

### `JWT_SECRET` 생성

```bash
openssl rand -base64 48
```

⚠️ 회전하면 모든 사용자 즉시 로그아웃. ([보안.md](보안.md) "키 회전" 참고)

## OAuth (수동)

| 키 | 필수 | 설명 |
|---|------|------|
| `GOOGLE_OAUTH_CLIENT_ID` | Google SSO 쓰면 | Google Cloud Console → Credentials |
| `KAKAO_OAUTH_NATIVE_APP_KEY` | Kakao SSO 쓰면 | Kakao Developers → 네이티브 앱 키 |
| `KAKAO_OAUTH_CLIENT_SECRET` | Kakao SSO 쓰면 | Kakao Developers → 보안 (선택) |

⚠️ Kakao Client Secret이 비활성이면 비워둬도 됨.

## 환경변수 추가 절차

1. **결정**: 어디서 쓰나 (Spring `@Value` / `application.yml` `${X}`)
2. **로컬 추가**: `.env` 또는 IDE Run Configuration
3. **Railway 추가**: 대시보드 → Variables → Add
4. **이 문서 갱신**: 키 / 필수 여부 / 획득 방법
5. **자동 재배포**: Railway는 환경변수 변경 시 자동 재시작

## 비밀 관리

- ❌ Git에 커밋 X
- ❌ 클라이언트(.env)에 백엔드 secret 노출 X (운영용 secret은 Railway에만)
- ✅ 키 회전 시 즉시 갱신 (이전 키는 invalidate)
- ✅ 가능하면 환경별 분리 (현재는 단일 production)

## 키 회전 절차

### `JWT_SECRET`
1. 새 secret 생성 (`openssl rand -base64 48`)
2. Railway → Variables → 갱신
3. 자동 재배포 (~2분)
4. **모든 사용자 즉시 로그아웃** — 다음 API 호출 시 401 → 클라가 로그인 화면으로 (TODO: 자동화)

⚠️ 평일 새벽 또는 점검 공지 후 권장.

### OAuth Client Secret
- 발급처 (Kakao/Google)에서 새 secret 생성
- Railway 환경변수 갱신
- 자동 재배포
- 사용자 영향 없음 (기존 토큰 그대로 동작, 새 로그인 시도 시만 새 secret 사용)

## 로컬 개발 환경변수

```bash
# 권장: 프로젝트 루트에 .env (gitignore됨)
PGHOST=localhost
PGPORT=5432
PGDATABASE=otaku_dev
PGUSER=postgres
PGPASSWORD=postgres
JWT_SECRET=dev-only-secret-min-32-chars-please

# 또는 IDE Run Configuration에서 환경변수 설정
```

`./gradlew bootRun` 실행 시 위 값 사용.
